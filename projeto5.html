Projeto 5

Controlador de Tráfego
Inteligente

Desenvolver um controlador de tráfego inteligente utilizando o projeto RTL e a linguagem VHDL.

O que?

  • Neste projeto foi desenvolvido um controlador de tráfego utilizando o projeto RTL e a linguagem de descrição de hardware VHDL

    • Como?

  • Para modelar o funcionamento do tráfego, neste projeto foi seguido a etapa do projeto RTL.
  • Feito isso, transformar os diagramas feitos utilizando o projeto RTL em códigos utilizando VHDL
  • Simular e analisar os resultados.
  • Fazer a pinagem na placa de FPGA para verificar o funcionamento.
